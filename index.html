ProfeRank

Proferank is a web application that enables students from different universities in Puerto Rico to rate their professors based on the courses they teach. The application includes a login system with Google authentication, which allows users to securely log in using their Google accounts. Additionally, Proferank has a comment/like system that enables users to leave feedback and react to other users' reviews.

The application is built using popular technologies such as Django, TailwindCSS, vanilla JavaScript, and PostgresSQL, which enable users to navigate and interact with the application smoothly. Proferank is deployed on a Linode server, which involved setting up a Linux server, installing Apache, UFW, SSL certificates, and setting up a custom domain.

Overall, Proferank is a useful and straightforward application that helps students make informed decisions about their education and provides valuable feedback to professors to improve their teaching practices. With its login system with Google authentication and comment/like system, Proferank provides a secure and engaging platform for students to share their experiences and opinions about their professors.

Subreddit Analyzer

The Sentiment Analysis web tool is a side project I built using Flask, NLTK, and Matplotlib. It uses the VADER sentiment analysis library to analyze the sentiment of posts in a subreddit and generates graphs to visualize the results. The application is served by a Flask server, which ensures a seamless delivery of the tool's outputs to end-users. Overall, the Sentiment Analysis web tool is a simple yet effective application that provides valuable insights into the sentiment of online communities.
